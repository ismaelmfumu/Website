Automated Snowplow (2023)

The primary goal of our project is to design and configure a robot with the capability to effectively clear snow or debris within a designated area defined by black tape boundaries. The robot must be equipped to handle both moving and stationary objects obstructing its path, relocating them to a specified zone while adhering to the set boundaries. Additionally, the robot should demonstrate the ability to navigate around the restricted tape without breaching it and continue its operations within the designated area.

Crucially, the project aims to ensure that the robot avoids incurring penalties outlined in the project requirements. These penalties encompass constraints such as constructing the robot within specified size limits, avoiding collisions with obstacles, and preventing the robot from crossing the black tape. In summary, the primary objective is to develop a robot that can proficiently clear debris within a confined space while strictly adhering to the stipulated rules and avoiding penalties.

Smart Garden Irrigation System (2023)

The smart irrigation system integrates hardware and software components to create an efficient and environmentally conscious solution for managing water resources in agriculture and landscaping. The system's hardware includes sensors that measure soil moisture levels and an Arduino Due microcontroller, which controls the irrigation system based on the sensor data. This setup ensures that plants receive the optimal amount of water, avoiding both over-watering and under-watering.

On the software side, the system features a web application that allows users to remotely monitor and control the irrigation system. Through the web app, users can access real-time data from the sensors, such as soil moisture levels, and adjust watering schedules or settings as needed. This remote access is particularly beneficial for farmers and gardeners who may not always be on-site to manage the irrigation system manually.

Additionally, the system utilizes a database to store and retrieve sensor data and user settings. This data can be used for analysis and optimization of watering schedules, as well as for historical tracking of moisture levels and watering patterns. By leveraging this information, users can make informed decisions about their watering practices, leading to water conservation and improved plant health.
